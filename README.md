**NutriScan AI | GitHub + Demo Video**

NutriScan AI is an intelligent system that combines data analysis and machine learning to analyze fruits and vegetables. The system:

Classifies produce as ripe, unripe, or rotten

Provides nutritional information for identified items

Helps users make healthier food choices and reduce food waste
